# APelCinc

## Basic setup

### Server

Go to folder `/server` and execute:

```
npm install
```

The server can be run with:

```
npm run start
```

### Webapp

Go to folder `/webapp` and execute:

```
npm install
```

The webapp can be run in different modes:

- Development

```
npm run dev
```

- Production

```
npm run build
```
